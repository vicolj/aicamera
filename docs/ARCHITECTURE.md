# 系统架构总览与里程碑

> v0.2 · **对齐海马爸比** · 全本地 · 零云存

## 1. 端到端数据流

```mermaid
flowchart TB
    subgraph Device["设备端 T31/T41"]
        CAM[105° 摄像头 + 940nm]
        MIC[麦克风]
        AI[全本地 AI]
        VENC[H.265]
        SD[(microSD FAT32)]
        P2P[P2P 服务]

        CAM --> VENC
        CAM --> AI
        MIC --> AI
        VENC --> SD
        VENC --> P2P
        AI -->|事件 JSON| P2P
    end

    subgraph Signal["轻量信令（无视频）"]
        PUSH[APNs/FCM]
        NAT[P2P 打洞协调]
    end

    subgraph Client["Flutter App"]
        APP[家长/分享账号]
    end

    P2P <-->|加密视频流| APP
    AI -.->|仅 type+ts| PUSH
    PUSH --> APP
    NAT -.-> P2P
    APP -.->|回看请求| P2P
    P2P -.->|从 SD 读流| APP
```

**与海马爸比一致**：视频 **设备 → 手机 P2P** 或 **设备 → SD**；云端不参与媒体路径。

## 2. 关键设计决策（v0.2）

| 决策 | 选择 | 对标 |
|------|------|------|
| SoC | 君正 T31 / **T41** | 海马爸比 T31X / T41 |
| 隐私 | **零云存 + 硬件禁止上传** | 三大隐私屏障 |
| AI | 遮脸+趴睡+呼吸+哭声+围栏 | 二代全功能 |
| 连接 | P2P + 推送信令 | 私有 App，不经云看视频 |
| 原型 | **iPhone 12** 验证 AI/App | 竞品无，我方加速手段 |
| Pro | 云台人形追踪 + 4K | Pro 版 |
