# EdgeRec Alert — 产品需求 (PRD)

> v1.0 · 边缘录播 + AI 告警 · To 集成商 / OEM

## 1. 产品定位

**EdgeRec Alert** 是可刷机/预装的 Linux 边缘软件包：接入网络摄像机，完成本地录播、AI 事件检测与告警上报，并提供 Qt 配参与 Web 回放。

**客户**：安防集成商、弱电工程、OEM 盒子小厂（他们面对终端客户，你面对集成商技术负责人）。

**不做**：自研硬件品牌、手机 App、公有云存、GB28181（v1）、人脸识别。

## 2. 版本规划

| 版本 | 平台 | 能力 |
|------|------|------|
| 社区版 | x86 / RV1126 | 1 路 RTSP 录播，无 AI |
| **Pro v1** | RV1126 | 4 路录播 + 1 路 AI 区域入侵 + Qt + Web |
| Pro x86 | x86 | 8 路录播 + 2 路 AI（后续） |

## 3. Pro v1 功能

| ID | 功能 | 说明 |
|----|------|------|
| F01 | RTSP 接入 | 2–4 路，H.264/H.265 |
| F02 | ONVIF 发现 | 可选，简化配网 |
| F03 | 本地录播 | MP4 分段，按天/通道目录 |
| F04 | 循环覆盖 | 按容量或保留天数 |
| F05 | AI 区域入侵 | RKNN，ROI 多边形 |
| F06 | 告警输出 | HTTP Webhook + GPIO（可选） |
| F07 | Qt 配置 | 通道、录像计划、告警区域画框 |
| F08 | Web 回放 | 局域网 HLS/MP4 时间轴 |
| F09 | 系统守护 | systemd 服务 + 日志 |

## 4. v1 明确不做

- 手机 App、云存 SaaS、GB28181
- 人脸识别、车牌、超过 1 种 AI 算法
- 多租户运维平台
- 全国现场实施（仅远程协助）

## 5. 非功能需求

| 项 | 指标 |
|----|------|
| 录播 | 4 路 1080p@15fps 稳定 72h |
| 告警延迟 | 事件触发 → Webhook < 2s |
| 存储 | eMMC/SD，循环策略可配置 |
| 配置 | Qt 断网可本地改参 |
