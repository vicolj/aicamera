# AI 婴儿看护摄像头 — 产品需求文档 (PRD)

> 版本：**v0.2** · 对齐参考：**海马爸比二代/Pro** · 目标：0–8 岁婴幼儿家庭

## 0. 设计原则（对齐海马爸比）

| 原则 | 说明 |
|------|------|
| **全本地 AI** | 所有视频分析在设备端完成，硬件层禁止上传云端 |
| **全本地存储** | 录像仅存 microSD，不做云存订阅 |
| **卧室隐私** | 画面打码、账号权限、数据不出设备 |
| **场景化看护** | 睡眠 / 客厅 / 分床 三模式，而非单一监控 |

**产品 slogan（内部）**：不是监视器，不做隐私泄密者。

---

## 1. 产品定位

面向新手父母的 **AI 智能婴儿看护器**：减轻育儿负担，在卧室场景提供 **遮脸、趴睡、哭声、睡眠分析** 等专业看护能力。

**核心价值**

| 价值 | 说明 |
|------|------|
| 安心 | 蒙头、趴睡、哭声、越界即时告警 |
| 省心 | 睡眠报告、精彩瞬间，减少手动记录 |
| 隐私 | 本地 AI + 本地 SD，从硬件阻断泄密 |

---

## 2. 用户与场景

**目标用户**

- 主用户：0–8 岁婴幼儿家长
- 分享用户：祖辈等（最多 5 个分享账号，权限可配）

**三大看护模式**（对齐海马爸比）

| 模式 | 适用年龄 | 核心能力 |
|------|----------|----------|
| **睡眠看护** | 0–12 个月 | 遮脸、趴睡、呼吸感知、睡眠报告 |
| **客厅看护** | 1–4 岁 | 虚拟围栏、人形移动侦测 |
| **分床看护** | 4–8 岁 | 深度睡眠分析、分床焦虑期记录 |

## 4. 非功能需求

| 类别 | 指标 |
|------|------|
| 哭声识别 | 准确率 ≥ 95%，误报 ≤ 3 次/夜 |
| 遮脸/趴睡 | 漏检率 < 5%，持续 3s 才告警 |
| 视频延迟 | 局域网 P2P P99 < 1s |
| 告警延迟 | 检测 → App 推送 < 3s |
| 稳定性 | 连续运行 72h 无崩溃 |
| 隐私 | **零视频上云**；仅推送元数据（事件类型+时间） |
| 存储卡 | 最大 128GB，**仅 FAT32** |
| 功耗 | 整机 < 5W |
| 材料 | RoHS，婴儿房安全涂层 |
| 认证 | SRRC、RoHS；CE 可选 |

---

## 5. 隐私设计（三大屏障 — 对齐海马爸比）

| 屏障 | 实现 |
|------|------|
| **数据本地处理** | AI 推理 + 录像均在设备；固件禁止向公网 URL 上传视频流 |
| **多重账号权限** | 分享账号可限制：仅看直播 / 不可回看 / 不可对讲 |
| **专属画面打码** | 用户框选成人活动区域打码；直播、回看、精彩瞬间均生效 |

---

## 6. 告警策略

| 级别 | 类型 | 行为 |
|------|------|------|
| 🔴 紧急 | 遮脸、趴睡、持续尖锐哭 | 强推送 + 设备端可选声音 |
| 🟠 重要 | 一般哭声、越界 | 推送 + 本地事件标记 |
| 🟡 提示 | 翻身、醒来 | 时间线记录；夜间可静默 |
| ⚪ 信息 | 入睡、睡眠报告 | 仅记录/日报 |

**智能安抚（F04）**：🟠 哭声触发后，设备自动播放预设白噪音/摇篮曲 3min，家长 App 可远程关闭。

---

## 7. 约束与假设

- 安装：专用支架，俯拍婴儿床，水平视角约 **105°**
- 网络：2.4GHz WiFi（必支持）；远程查看走 P2P，**不经云端存视频**
- 产品尺寸参考：约 132×133×103mm
- 目标零售价：**基础版 ¥399–499 · Pro 版 ¥599–799**
- BOM 目标：基础版 ≤ ¥200，Pro（含云台）≤ ¥280

---

## 8. 验收标准（MVP）

1. 哭声检测召回率 ≥ 95%，电视/说话声误报 ≤ 3 次/夜
2. 遮脸：口鼻遮挡 > 3s 告警；趴睡：侧角 > 70° 告警
3. 睡眠报告与人工记录偏差 < 15min/天
4. SD 卡 128GB FAT32 连续录像 7 天无异常
5. 断网时：本地 AI + 录像 + 局域网预览仍可用
6. **安全测试**：抓包验证无视频数据离开局域网（除 P2P 直连家长手机）
