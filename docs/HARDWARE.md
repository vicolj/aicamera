## 3. 主控选型 — 君正 T 系列（对齐海马爸比）

### 3.1 基础版 — Ingenic T31ZX

| 项 | 参数 |
|----|------|
| 对标 | 海马爸比一代/二代（T31X 系列） |
| 分辨率 | 1920×1080 @ 30fps |
| NPU | ~0.8 TOPS INT8 |
| 内存 | 内置 128MB DDR |
| 优势 | 成本低、功耗低、IPC 供应链成熟 |

### 3.2 Pro 版 — Ingenic T41ZN（主推荐量产）

| 项 | 参数 |
|----|------|
| 对标 | 海马爸比 Pro / 4 代 |
| CPU | Dual-core MIPS XBurst2 @ 1.4GHz |
| NPU | **1.2 TOPS** INT8 |
| 视频 | **4K** 编码，2160P@30fps |
| ISP | Tiziano 升级版，夜视 WDR |
| 内存 | 内置 128MB DDR3 |
| 选型理由 | 与海马爸比同平台，算力够跑呼吸感知+多模型 |

> **变更说明**：v0.1 选用 RV1106，v0.2 改为 **君正 T 系列** 以对齐竞品平台与算力。

## 4. 三轨开发策略

| 轨道 | 硬件 | 用途 | 周期 |
|------|------|------|------|
| **A. iPhone 12** | 闲置手机 + 支架 | AI 算法、App、睡眠/哭声模型 | 2–4 周 |
| **B. 树莓派 5** | Pi + Camera Module 3 | 联调 P2P、SD 回放逻辑 | 可选 |
| **C. T41 开发板** | 君正 IdeaT41 / 模组 | 量产固件、ISP 夜视、云台 | 6–8 周 |

**推荐路径**：iPhone 12 验证 AI → T41 工程样机 → 量产。

### 4.1 iPhone 12 原型轨（轨道 A）

| 能验证 | 不能验证 |
|--------|----------|
| 哭声/遮脸/趴睡算法 | 940nm 夜视 |
| App 预览、告警、打码 UI | 云台追踪 |
| Core ML 模型迭代 | T41 ISP 调校 |
| 睡眠/呼吸感知原型 | 专用外形与 SD 循环录 |
