# 硬件选型与 BOM

> 对应 PRD v0.1 · 双轨策略：**原型验证轨** + **量产轨**

## 1. 总体架构

```
                    ┌─────────────────────────────────────┐
                    │           婴儿看护摄像头             │
                    │                                     │
  5V/2A ──► PMIC ──► SoC ──► ISP ──► 镜头+Sensor         │
                    │   │                                 │
                    │   ├── NPU (边缘 AI)                  │
                    │   ├── WiFi/BLE                       │
                    │   ├── I2S ◄── MEMS 麦克风            │
                    │   ├── I2S ──► Class-D 功放 ──► 喇叭  │
                    │   ├── SDMMC ◄── microSD              │
                    │   └── GPIO ◄── 温湿度 (可选)          │
                    │                                     │
                    │   IR-CUT + 940nm IR LED 阵列          │
                    └─────────────────────────────────────┘
```

## 2. 方案对比

| 维度 | A. 原型轨（开发） | B. 量产轨（产品） |
|------|-------------------|-------------------|
| 主控 | Raspberry Pi 5 (4GB) | Rockchip **RV1106G** |
| Sensor | Pi Camera Module 3 | **GC4653** 或 IMX415 |
| AI 算力 | CPU + Hailo-8L（可选） | 内置 NPU ~0.5 TOPS |
| WiFi | USB  dongle / 板载 | RTL8723 / AIC8800 模组 |
| 成本 | ~¥600（单台开发） | BOM ~¥120–180 |
| 用途 | 算法/App/后端联调 | 结构/认证/量产 |

**推荐路径**：先用 **方案 A** 跑通软件栈，再移植到 **方案 B** 工程样机。

## 3. 量产轨详细选型（方案 B）

### 3.1 主控 SoC — Rockchip RV1106G

| 项 | 参数 |
|----|------|
| CPU | ARM Cortex-A7 @ 1.2GHz |
| NPU | 0.5 TOPS，支持 INT8 |
| ISP | 内置，支持 WDR、3DNR |
| 视频 | H.264/H.265 编码 2K@30fps |
| 接口 | MIPI CSI、I2S、SDIO、USB |
| SDK | RK IPC SDK / Buildroot |
| 选型理由 | 婴儿看护 AI 算力足够、IPC 生态成熟、成本可控 |

**备选**

| SoC | 优点 | 缺点 |
|-----|------|------|
| Ingenic T41ZN | 国内 IPC 常用、价格低 | NPU 生态弱于 RK |
| SigmaStar SSC377 | 夜视 ISP 好 | SDK 文档偏封闭 |
| ESP32-S3 | 极低成本 | 无法跑 2K + 多模型 AI |

### 3.2 图像传感器 — GC4653（主选）

| 项 | 参数 |
|----|------|
| 分辨率 | 4MP (2560×1440) |
| 帧率 | 30fps |
| 接口 | MIPI CSI-2 2-lane |
| 低照 | 适合红外夜视场景 |
| 备选 | Sony IMX415（更高动态范围，贵 ~¥15） |

### 3.3 镜头

| 项 | 规格 |
|----|------|
| 焦距 | 2.8mm 或 3.6mm 定焦 |
| 视角 | 水平 100°–110°（覆盖婴儿床） |
| 光圈 | F2.0 |
| 结构 | M12 座 + 金属镜筒，防松动 |
| IR | 支持 Day/Night IR-CUT 切换 |

### 3.4 夜视模组

| 组件 | 规格 |
|------|------|
| IR-CUT | 电磁式，ISP 控制 |
| 补光灯 | 940nm IR LED × 4–6，总功率 < 1.5W |
| 说明 | 940nm 无红爆点，不干扰婴儿睡眠 |

### 3.5 音频

| 组件 | 型号建议 | 用途 |
|------|----------|------|
| 麦克风 | INMP441 × 1 或 MSM261S4030H0 | 哭声采集 |
| 功放 | MAX98357A | I2S 输入 Class-D |
| 喇叭 | 8Ω 1W 腔体喇叭 | 对讲 / 白噪音（V2） |
| 算法 | AEC（回声消除）+ NS（降噪） | 双向对讲必备 |

### 3.6 连接

| 组件 | 规格 |
|------|------|
| WiFi | 802.11 b/g/n/ax **2.4GHz 必支持** |
| BLE | 5.0，用于配网（BluFi / 自定义 GATT） |
| 天线 | PCB 天线或 IPEX 外置（结构允许时外置增益更好） |

### 3.7 存储与电源

| 组件 | 规格 |
|------|------|
| Flash | 128Mb SPI NOR（系统 + 模型） |
| RAM | 256MB DDR3（RV1106 标配） |
| SD | microSD UHS-I，Class 10，最大 256GB |
| 电源 | USB-C 5V/2A 输入，DC-DC 3.3V/1.2V |
| 保护 | TVS、过流、看门狗 GPIO |

### 3.8 可选传感器（V2 / 高配版）

| 传感器 | 型号 | 用途 |
|--------|------|------|
| 温湿度 | SHT30 / AHT20 | 房间环境告警 |
| 毫米波 | TI IWR6843 / 国产 60GHz | 非接触呼吸心率 |
| PIR | BS612 | 低功耗触发（电池版才需要） |

## 6. 机械与安装

```
        墙面/支架
            │
            ▼
    ┌───────────────┐
    │   摄像头本体    │  ← 俯角 30°–45°
    └───────┬───────┘
            │
    ════════╪════════  婴儿床护栏
            │
         [  婴儿  ]
```

| 要求 | 规格 |
|------|------|
| 安装 | 桌面夹持 + 墙装支架（包材标配） |
| 俯角 | 可调 ±30° |
| 线缆 | 3m USB-C 电源线（床头插座） |
| 安全 | 无小零件脱落风险；圆角外壳；符合 GB 6675 邻接要求 |
